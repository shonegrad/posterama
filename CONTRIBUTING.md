# Contributing

Thanks for the interest in contributing to Posterama.

## Setup
- Node LTS
- `npm install`
- `npm run dev`

## How to contribute
- Open an issue first for non-trivial changes.
- Keep PRs focused.
- Include:
  - what changed
  - why
  - how to test

## Quality bar
- Lint passes
- Typecheck passes
- Tests pass (or explain why none are added)
- No secrets introduced
